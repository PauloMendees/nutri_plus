# Step 02.1 - User Sync Flow

## Goal

Synchronize Supabase users with local database.

## JWT Data

Extract:

- sub
- email
- name

## Mapping

sub -> authProviderId

authProvider -> SUPABASE

## Create User

If user does not exist:

- create User
- create NutritionistProfile or PatientProfile

## Update User

If user exists:

- update name
- update email

## Referral Code

Optional for patients.

If supplied:

1. Find nutritionist by referralCode
2. Assign nutritionistId
